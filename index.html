如何加強自己學習的動力？

Posted on 2020-03-07 In 生活紀錄

前陣子撞牆期有點久，過了快半年我才釐清自己的狀態其實是因為找不到確切的方向。突破了這點後，發現自己又有前進的動力了！我想把這段時間的經驗寫下來，希望能幫助到有類似困惑的人。

投入前端學習的這段時間，我發現我時間管理實在很差。不過，我後來意識到這並不是時間管理的問題，是我完全無法投入學習的過程，很容易在一個小章節就馬上分心。這並不是我原本的狀態。在上一份工作中，我可以全心投入工作過程，讓自己可以在十分鐘甚至是五分鐘之內結束一份小任務，這不是因為我時間管理特別好，是因為我把注意力全心全意地投入在工作的過程中。我很喜歡那種感覺。可是，在離職之後，儘管一開始我還是可以保有原本的投入狀態，當時學習的品質也很好。可是，大約過了半年之後，我發現我忽然沒辦法進入那樣的狀態了！

經過一段時間摸索，我才意識到原來我是找不到更明確的方向。儘管我有成為前端工程師，這樣的目標。但是，我發現這樣的目標對我來說，其實非常模糊，難以想像、捉摸。成為前端工程師，是怎樣的前端工程師呢？只是完成這些課程的作業、多做幾個作品嗎？像這樣的問題一直在我腦海裡出現。我總覺得這樣的目標對我來說還是太模糊。

vue cli中使用bootstrap的JS元件

Posted on 2020-03-07 In 前端技術筆記

一般來說，如果要在 Vue cli 中使用 bootstrap 的 modal （或是其他 JS 元件），只要在 Vue cli 中的進入點，main.js 這支檔案中，匯入 bootstrap 即可。在這個部分，Vue cli 可能會提醒你要安裝 jquery 和 popper.js 兩支檔案，因為 bootstrap 的 JS 元件就是用這兩個函式庫寫成的。這時候，只要跟著 npm 的指令做就可以了。

安裝了 jqeury，也把 bootstrap 匯入 Vue 專案中後，我們就可以把 modal 這個元件寫到 Vue 元件的 template 部分裡，不需要額外寫其他程式碼，這個 modal 就會運行了。這部分和我們一般在寫 bootstrap 時一樣。
